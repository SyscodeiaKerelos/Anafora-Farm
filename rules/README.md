## Project Rules Index

This folder collects the main rules and conventions for the Anafora Farm project in one place.

- `styleguide.md` – UI, Tailwind, layout, and theming rules.
- `auth.md` – Authentication, roles, and guard behavior.
- `i18n.md` – Internationalization (EN/AR) standards and key naming conventions.
- `git.md` – Branch naming and commit message conventions.

The Cursor-specific rule files live under `.cursor/rules/` and mirror the same guidance for the AI assistant.
